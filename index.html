PROMPT 1

Merge — propose the template (attach both files)

You are a careful, methodical data analyst. I'm attaching two CSV exports covering the same six months: cycle_test_data.csv from my period app (columns: date, period_flow, symptoms, mood - only the days I logged something) and wearable_test_data.csv from my wearable (columns: date, sleep_hours, resting_hr_bpm, hrv_ms, skin_temp_c - one row per night I wore it).

Review both files. Then propose a structure for ONE merged table: which columns it should have and how you would combine the two files. Merge on date, and give every calendar day from the first to the last date its own row - even days that appear in neither file.

Show me the proposed column names and 3 example rows as a template first, and wait for my OK. Don't build anything yet.

PROMPT 2

Merge — build the file

I approve the template - build it. Fill the merged table from the two files using real code, matching rows only on exact dates. If a file has no value for a day, leave that cell blank - never invent, estimate or interpolate values.

Then give me the merged table as a downloadable file (Excel or CSV), and tell me: how many days it covers, how many days have no period-app entry, and how many nights have no wearable data.

PROMPT 3

Cycle days & phases

Now, using the merged table and real calculations (never estimates):
1. Find the first day of each cycle (a period day that follows days without a period).
2. Number every day within its cycle (day 1, day 2, ...).
3. Label every day with a cycle phase: menstrual (days 1-5), follicular, ovulatory (around 14 days before the next period), luteal.

Show me the first 10 rows with cycle day and phase, and tell me how many cycles you found and how long each one was.

PROMPT 4

Averages & charts by phase

Now, with real calculations:
1. Make a table with the average skin temperature, resting heart rate, HRV and sleep hours for each of the four phases.
2. Create one chart per signal: skin temperature by cycle day (all cycles on top of each other), and the phase averages for resting heart rate and HRV.
3. Then zoom in: compare the last 2 days before each period with the rest of the luteal phase. What happens to HRV, resting heart rate and sleep on those days?
Explain each chart in one sentence.

PROMPT 5

Plain-language takeaways

In plain language, no technical terms:
1. What does my data say about how my body changes across my cycle?
2. Is there any stretch in these six months where something unusual happened? If yes: when, what do you see in the data, and what everyday explanations could fit?
3. What is one thing I could pay attention to in my next cycle?
